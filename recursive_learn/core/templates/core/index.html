Una breve introducción

Pincha en cualquiera de las cajas para obtener más información de cada una de las etapas de un ejercicio recursivo.

Entradas

Salidas

[3, 2, -6, 8, 11]

Entrada

El primer paso siempre es elegir un tamaño del problema, el cual es el elemento que debemos reducir para generar subproblemas. También tendremos que elegir el/los casos base. El caso base es el elemento de la condición de parada, momento en el cual la resolución del problema es trivial.

Pincha fuera para cerrar

Descomposición

[2, -6, 8, 11]

Subproblema

El subproblema se obtiene de la descomposición de la entrada. Con el tamaño del problema debemos decidir como se va a reducir, por ejemplo, en una lista podriamos quitar el primer elemento. Al descomponer nos quedará uno o varios elementos que utilizaremos más adelante para resolver el problema.
A veces al elegir una descomposición los casos base podrían no ser suficientes. Por ejemplo, si quitamos elementos de la lista de 2 en 2 no valdría con solo llegar a una lista vacia, también podría quedar un único elemento. Estas descomposiciones generan errores de ejecución o recursión infinita si no se contemplan todos los casos base.

Pincha fuera para cerrar

Llamada recursiva

2 + (-6) + 8 + 11 = 15

Solución simple

Una vez hacemos la llamada recursiva vamos a asumir que el subproblema se resuelve, no tenemos que plantearnos que pasa en las sucesivas llamadas recursivas. Tras dar este salto de fe solo nos faltará un paso.

Pincha fuera para cerrar

Derivación caso recursivo

3 + 2 + (-6) + 8 + 11 = 18

Solución

La solución la obtendremos de operar el resultado del subproblema con el valor de la descomposición. Tenemos que pensar que operación debemos realizar para obtener la solución final utilizando ambos valores.
